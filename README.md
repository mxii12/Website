# Neil Amirtha — Personal Website

A slick, single-file personal website for Neil Amirtha.

## Stack

- Vanilla HTML, CSS, and JavaScript — no build tools, no npm, no frameworks
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- Deployable to any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages)

## Features

- Interactive dot-grid canvas background with mouse-pull physics
- Text scramble animation on hero heading
- Scroll-triggered fade-in reveals (IntersectionObserver)
- Frosted-glass nav with active section tracking
- Gradient card borders with glow hover effects
- Infinite CSS marquee for tech stack
- Email copy-to-clipboard
- Custom cursor trailer
- Fully responsive — mobile hamburger menu
- Accessible: WCAG AA+ contrast, `prefers-reduced-motion` support

## Usage

Open `index.html` in any browser. No server required.
